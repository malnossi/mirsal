P2P File Transfer Made Beautiful.

Mirsal is a zero-knowledge secure cross-platform desktop application designed to stream files and directories between devices in real-time. Powered by Go, Wails v2, and the magic of peer-to-peer security.

Key Capabilities

Zero-Knowledge

Uses PAKE (Password Authenticated Key Exchange) encryption keys to negotiate peer connections. Files stream directly—no servers log or store content.

Auto-Folder Compression

Drop a directory and Mirsal automatically packs it into a temporary archive on-the-fly. The receiver seamlessly extracts the folder and cleans up temporary items.

Event Bridge Calculations

Monitors speed in real-time, displaying elapsed percentages and estimating seconds remaining via high-feedback circular and linear progress displays.
